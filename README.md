## What I'm building right now

### Focus OS — Attention Intelligence Platform
> *A Chrome Extension + web dashboard that tracks your cognitive behavior and tells you the truth about your focus.*

Most productivity tools track **time**. Focus OS tracks **quality of attention** — tab switch frequency, distraction spikes, idle patterns, and context-switching cost. Every session gets a 0–100 focus score. Every week gets an AI-generated cognitive report.

**Why it's not a typical project:**
- Chrome Extension MV3 with background service workers — privileged browser context most devs never touch
- Custom focus scoring algorithm (weighted: session depth 40%, switch frequency 35%, distraction ratio 25%)
- Real-time event streaming via WebSockets — dashboard updates as you work
- BullMQ scheduled job generates weekly AI report via OpenAI API
- Redis for session caching, Docker for deployment

**Stack:** MongoDB · Express · React · Node.js · Chrome Extension MV3 · Socket.io · Redis · BullMQ · OpenAI API · Docker
